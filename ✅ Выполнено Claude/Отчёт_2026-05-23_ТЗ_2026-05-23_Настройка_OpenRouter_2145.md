# Настройка OpenRouter для Claude Code

## 📋 Пошаговый конспект

### Шаг 1: Регистрация на OpenRouter
1. Перейти на https://openrouter.ai
2. Создать аккаунт
3. Пополнить баланс (рекомендуется начать с $5-10)

### Шаг 2: Получение API-ключа
1. Зайти в личный кабинет OpenRouter
2. Перейти в раздел "API Keys" или "Keys"
3. Создать новый ключ (кнопка "Create Key")
4. Скопировать ключ формата `sk-or-v1-...` (он показывается только один раз!)

### Шаг 3: Настройка терминала на Макбуке
1. Открыть терминал
2. Выполнить команду: `nano ~/.zshrc`
3. Прокрутить в самый низ файла (клавиши ↓)
4. Вставить три строки:
   ```bash
   export ANTHROPIC_BASE_URL="https://openrouter.ai/api"
   export ANTHROPIC_API_KEY=""
   export ANTHROPIC_AUTH_TOKEN="ваш_ключ_от_openrouter"
   ```
5. Заменить `ваш_ключ_от_openrouter` на скопированный ключ
6. Сохранить: `Ctrl + O`, затем `Enter`
7. Выйти: `Ctrl + X`
8. Применить изменения: `source ~/.zshrc`

### Шаг 4: Проверка работы
1. Перейти в папку проекта
2. Запустить `claude`
3. Убедиться, что соединение установлено

## 📋 Детали

### Принцип работы
- **Claude Code** — бесплатный клиент-интерфейс
- **OpenRouter** — платформа-посредник для доступа к AI-моделям
- Запросы перенаправляются с вашего компьютера через OpenRouter к модели Claude
- Оплата списывается **только** с баланса OpenRouter

### Важные технические моменты
1. **Base URL**: `https://openrouter.ai/api` (без `/v1` в конце)
2. **API_KEY оставляем пустым** — чтобы избежать конфликтов
3. **AUTH_TOKEN** — сюда вставляется ключ от OpenRouter
4. Файл конфигурации: `~/.zshrc` (для оболочки zsh на macOS)

### Что происходит после настройки
- При каждом запросе Claude Code отправляет данные на OpenRouter
- OpenRouter видит ваш ключ и списывает токены с вашего баланса
- Anthropic не получает от вас никаких платежей
- В личном кабинете OpenRouter вы видите историю расходов

## 💡 Рекомендации

### Безопасность
- **Никогда не публикуйте** API-ключ в открытых репозиториях
- Создайте отдельный ключ для каждого проекта (в OpenRouter можно создавать несколько)
- Регулярно проверяйте баланс и расходы в личном кабинете

### Финансы
- Начните с минимального пополнения $5-10
- Отслеживайте расход токенов в дашборде OpenRouter
- Модель Claude 3.5 Sonnet стоит примерно $3 за 1М входных токенов и $15 за 1М выходных

### Оптимизация
- Используйте более дешевые модели для простых задач (через настройки OpenRouter)
- Сохраняйте контекст диалога для экономии токенов
- Периодически очищайте историю, чтобы не передавать лишний контекст

### Проверка настройки
После выполнения всех шагов проверьте командой:
```bash
echo $ANTHROPIC_BASE_URL
echo $ANTHROPIC_AUTH_TOKEN
```
Должны вывестись URL и ваш ключ.

# 📝 Конспект: Настройка Claude Code через OpenRouter

## 🎯 Суть задачи
Перенаправить все запросы из **Claude Code** на сервис **OpenRouter**, чтобы оплата шла только с баланса OpenRouter, без необходимости регистрации в Anthropic.

---

## 📋 Пошаговая инструкция

### Шаг 1: Регистрация на OpenRouter
1. Перейти на https://openrouter.ai
2. Зарегистрироваться / войти в аккаунт
3. Пополнить баланс (например, $5-10 для начала)

### Шаг 2: Получение API-ключа
1. Зайти в раздел **API Keys** (или **Settings → API Keys**)
2. Создать новый ключ (кнопка **Create Key**)
3. Скопировать ключ формата `sk-or-v1-...`
4. Сохранить его в безопасном месте

### Шаг 3: Настройка терминала на Макбуке
1. Открыть терминал
2. Выполнить команду:
   ```bash
   nano ~/.zshrc
   ```

3. В конце файла добавить три строки:
   ```bash
   # Перенаправляем Claude Code на OpenRouter
   export ANTHROPIC_BASE_URL="https://openrouter.ai/api"
   export ANTHROPIC_API_KEY=""
   export ANTHROPIC_AUTH_TOKEN="sk-or-v1-ВАША_СТРОКА_КЛЮЧА_СЮДА"
   ```

4. Сохранить файл: **Ctrl + O** → **Enter**
5. Выйти из редактора: **Ctrl + X**

### Шаг 4: Применение изменений
```bash
source ~/.zshrc
```

### Шаг 5: Проверка работы
1. Перейти в папку проекта
2. Запустить `claude`
3. Проверить, что запросы идут через OpenRouter (в личном кабинете OpenRouter будет виден расход токенов)

---

## ⚙️ Важные детали

- ✅ **URL без `/v1`**: `https://openrouter.ai/api` (именно так!)
- ✅ **ANTHROPIC_API_KEY** оставляем **пустым**
- ✅ Оплата **только** с баланса OpenRouter
- ✅ Никаких подписок Anthropic не требуется

### Долгосрочная стратегия

**Архитектура облачной работы:**

```yaml

    ↓ (только текст)

    ↓ (git push)

    ↓ (триггер)
GitHub Actions (тяжёлые процессы)
    ↓ (результаты)
GitHub Repository
    ↓ (git pull)
MacBook (получает готовое)
```

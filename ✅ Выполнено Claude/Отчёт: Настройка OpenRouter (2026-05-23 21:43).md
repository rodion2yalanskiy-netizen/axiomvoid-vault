# Конспект: Настройка OpenRouter для Claude Code

## 📝 Пошаговый план действий

### Шаг 1: Регистрация на OpenRouter
1. Перейти на сайт https://openrouter.ai
2. Создать аккаунт
3. Пополнить баланс (рекомендуется начать с $5-10)

### Шаг 2: Получение API-ключа
1. Войти в личный кабинет OpenRouter
2. Перейти в раздел "API Keys"
3. Создать новый ключ (кнопка "Create Key")
4. Скопировать ключ формата `sk-or-v1-...`
5. Сохранить его в безопасном месте

### Шаг 3: Настройка терминала на MacBook
1. Открыть терминал (Terminal.app)
2. Выполнить команду: `nano ~/.zshrc`
3. Прокрутить в конец файла (стрелками вниз)
4. Вставить следующие строки:
   ```bash
   # Перенаправляем Claude Code на OpenRouter
   export ANTHROPIC_BASE_URL="https://openrouter.ai/api"
   export ANTHROPIC_API_KEY="" 
   export ANTHROPIC_AUTH_TOKEN="ваш_ключ_от_openrouter_сюда"
   ```
5. Заменить `ваш_ключ_от_openrouter_сюда` на реальный ключ
6. Сохранить: `Ctrl + O`, затем `Enter`
7. Выйти: `Ctrl + X`
8. Применить изменения: `source ~/.zshrc`

### Шаг 4: Проверка настройки
1. Перейти в папку проекта
2. Запустить `claude`
3. Проверить работу через тестовый запрос

## 📋 Детали

### Принцип работы системы
**Claude Code** → **OpenRouter** → **Claude 3.5 Sonnet**

- Claude Code — бесплатный клиент-интерфейс
- OpenRouter — платный прокси-сервис (оплата по факту использования)
- Списание средств происходит **только с баланса OpenRouter**
- Никаких дополнительных подписок не требуется

### Важные технические моменты
1. **URL без `/v1`**: Используется `https://openrouter.ai/api` (не `/api/v1`)
2. **Пустой ANTHROPIC_API_KEY**: Специально оставляется пустым для избежания конфликтов
3. **Переменная окружения**: Ключ прописывается в `ANTHROPIC_AUTH_TOKEN`
4. **Файл конфигурации**: Используется `~/.zshrc` (для zsh на современных macOS)

## 💡 Рекомендации

### Безопасность
1. **Никогда не публикуйте** API-ключ в открытых репозиториях
2. Храните резервную копию ключа в менеджере паролей (1Password, Bitwarden)
3. Периодически обновляйте ключи (создавайте новые, удаляйте старые)

### Мониторинг расходов
1. Настройте лимиты расходов в личном кабинете OpenRouter
2. Регулярно проверяйте графики использования
3. Начните с небольшого баланса ($5) для тестирования

### Оптимизация работы
1. Закрывайте сессии Claude Code после завершения работы
2. Используйте краткие, четкие промпты для экономии токенов
3. Мониторьте баланс перед началом больших задач

### Альтернативный файл конфигурации
Если используете bash вместо zsh, замените `~/.zshrc` на `~/.bash_profile` или `~/.bashrc`

### Проверка текущих настроек
Команда для проверки переменных окружения:
```bash
echo $ANTHROPIC_BASE_URL
echo $ANTHROPIC_AUTH_TOKEN
```

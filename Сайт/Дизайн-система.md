## Анимации

| Переменная | Значение |
|---|---|
| `--transition-smooth` | `0.6s cubic-bezier(0.16, 1, 0.3, 1)` |
| `--transition-fast` | `0.3s cubic-bezier(0.16, 1, 0.3, 1)` |

## Правила разработки

- Только CSS-переменные, без хардкода цветов и шрифтов
- Адаптивность: 375 / 768 / 1440px
- Анимации только через `transform` + `opacity`
- Бренд: **Axiom:Void** (не QSnera, не QSNERA, не Aurelia)
- Scroll-reveal: класс `.reveal` + IntersectionObserver

# ТЗ: Добавление футера "Pipeline v1.0 — powered by AI"

## Описание задачи
Добавить в футер сайта premium-tiling-website текстовую строку: "Pipeline v1.0 — powered by AI".

## Требования к реализации
- **Визуальное оформление**:
    - Шрифт: Inter
    - Размер: мелкий
    - Цвет: #888 (серый)
- **Расположение**: в футере страницы
- **Результат**: Текст должен быть виден на странице.

## Процесс выполнения
1. Найти файл футера в проекте premium-tiling-website.
2. Добавить HTML-разметку с необходимым текстом.
3. Добавить CSS-стили для соответствия дизайну QSNera.
4. Проверить отображение.
5. Зафиксировать изменения (commit) и отправить в репозиторий (push).

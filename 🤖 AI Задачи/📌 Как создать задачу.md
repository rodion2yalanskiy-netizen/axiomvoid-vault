# 📌 Как создать задачу с телефона

**Просто создай новую заметку в этой папке** — и всё!

## Пример минимальной задачи:
```
Сделай анализ конкурентов в плиточном бизнесе
```

## Пример с деталями:
```
---
type: content
priority: high
---

Напиши 5 постов для Instagram про укладку плитки в ванной
Стиль: премиум, профессионально
Длина: 3-4 предложения каждый
```

## Что произойдёт автоматически:
1. Заметка → Mac (каждые 5 мин)
2. Mac → GitHub 
3. GitHub Actions → Claude выполнит задачу
4. Отчёт появится в папке **✅ Выполнено Claude**

⏱ Время выполнения: 5-15 минут

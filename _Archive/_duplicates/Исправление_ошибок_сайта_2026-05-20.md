2. **Как исправлено:** Глобальная замена `replace_all` по всему файлу `index.html`. Затронутый файл: `index.html`.
3. **Результат:** Единообразное и корректное написание `QSNera` на всей странице. Email-адрес `inquire@qsnera.com` оставлен в нижнем регистре — это норма для доменных имён.

### ⚠️ Категория 3 — Ошибки в коде / JS-логика

> **Общая суть проблемы:** В скриптах страницы отсутствовали проверки на существование DOM-элементов перед вызовом методов на них. Если браузер по любой причине не найдёт элемент (устаревший кэш, частичная загрузка, A/B тест, будущий рефакторинг HTML), весь JavaScript на странице сломается с ошибкой `TypeError: Cannot read properties of null`.

#### Ошибка 3.1: Кастомный курсор — нет null-guard
1. **Что было не так:** `cursorDot.style.left = ...` вызывался немедленно при каждом `mousemove` без проверки, существует ли элемент `#custom-cursor`. Падение блокировало весь остальной JS на странице.
2. **Как исправлено:** Весь блок обработчиков курсора обёрнут в `if (cursorDot && cursorOutline) { ... }`.
3. **Результат:** При отсутствии элементов курсора страница продолжает работать в штатном режиме.

#### Ошибка 3.2: Хедер — нет null-guard на scroll-обработчик
1. **Что было не так:** `header.classList.add('scrolled')` вызывался при каждом событии `scroll` без проверки `header !== null`.
2. **Как исправлено:** Блок обёрнут в `if (header) { ... }`.
3. **Результат:** Безопасная работа даже если `#header` будет переименован или удалён.

#### Ошибка 3.3: Слайдер отзывов — нет null-guard + дублирующийся код
1. **Что было не так:** `prevBtn.addEventListener(...)` и `nextBtn.addEventListener(...)` вызывались без проверок. Дополнительно: блок auto-rotate (`setInterval`) существовал **дважды** — один раз внутри нового null-guard блока и ещё раз снаружи, вне области видимости переменных, что вызывало `ReferenceError: showSlide is not defined` в runtime при запуске второго блока.
2. **Как исправлено:** Весь блок (slider + showSlide + auto-rotate + сброс таймера) перенесён внутрь единого `if (slides.length && prevBtn && nextBtn) { ... }`. Дублирующий внешний блок удалён.
3. **Результат:** Устранены сразу две ошибки: потенциальный `TypeError` и гарантированный `ReferenceError`. Код очищен от дублирования.

#### Ошибка 3.4: Форма обратной связи — нет null-guard
1. **Что было не так:** `form.addEventListener('submit', ...)` вызывался напрямую без проверки существования `form` и `feedback`. Если форма не отрендерилась, страница падала.
2. **Как исправлено:** Весь submit-обработчик обёрнут в `if (form && feedback) { ... }`. Попутно убрана вспомогательная переменная `inputs` — заменена инлайн-вызовом `.querySelectorAll().forEach()`.
3. **Результат:** Страница стабильна при любом состоянии DOM.

#### Ошибка 3.5: Мобильная навигация — нет null-guard
1. **Что было не так:** `menuToggle.addEventListener('click', ...)` и `navLinks.querySelectorAll(...)` вызывались без проверок. При мобильном breakpoint, если элементы не найдены — JS-ошибка.
2. **Как исправлено:** Весь блок (toggle + закрытие по клику на ссылку) обёрнут в `if (menuToggle && navLinks) { ... }`. Переменная `mobileLinks` устранена — заменена инлайн-цепочкой.
3. **Результат:** Мобильная навигация устойчива, код стал чище.

### JavaScript — исправление дублирующегося блока carousel
```javascript
// ДО: showSlide() и setInterval() объявлены внутри if-блока,
// но второй setInterval() вызывался СНАРУЖИ него — ReferenceError:
if (slides.length && prevBtn && nextBtn) {
  const showSlide = (index) => { /* ... */ };
  // ...
}
let carouselTimer = setInterval(() => showSlide(currentSlide + 1), 8000); // ❌ ReferenceError

// ПОСЛЕ: весь carousel-код сгруппирован внутри одного guard-блока:
if (slides.length && prevBtn && nextBtn) {
  const showSlide = (index) => { /* ... */ };
  prevBtn.addEventListener('click', () => showSlide(currentSlide - 1));
  nextBtn.addEventListener('click', () => showSlide(currentSlide + 1));

  let carouselTimer = setInterval(() => showSlide(currentSlide + 1), 8000);
  [prevBtn, nextBtn].forEach(btn => {
    btn.addEventListener('click', () => {
      clearInterval(carouselTimer);
      carouselTimer = setInterval(() => showSlide(currentSlide + 1), 8000);
    });
  });
}
```

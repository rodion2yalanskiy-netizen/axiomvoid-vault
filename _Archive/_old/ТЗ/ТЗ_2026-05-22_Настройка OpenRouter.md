# ТЗ: Настройка OpenRouter

## Задача из Obsidian
Вам **достаточно только второго варианта (OpenRouter)**. Деньги и токены будут списываться **исключительно с вашего баланса на OpenRouter**.
Никаких скрытых списаний, подписок или отдельных токенов внутри самого Claude Code (или плагинов в Obsidian) не будет.
Вот как это работает изнутри и почему вам не нужно платить дважды.
## 🧭 Как устроен этот механизм (Принцип «Трубы»)
Инструмент **Claude Code** — это бесплатная программа-клиент от компании Anthropic, которую вы устанавливаете на свой Макбук. Сама по себе она денег не просит, это просто интерфейс (инструмент командной строки).
По умолчанию Claude Code настроен так, чтобы отправлять запросы на серверы Anthropic. Но разработчики оставили возможность перенаправить эту «трубу» в любое другое место.
Когда мы настраиваем систему, мы меняем внутренний адрес отправки запросов (Base URL) на OpenRouter.
 1. Вы пишете команду в **Claude Code**.
 2. Программа думает, что отправляет запрос в Anthropic, но благодаря настройкам запрос улетает на https://openrouter.ai/api.
 3. **OpenRouter** принимает этот запрос, видит ваш секретный API-ключ (sk-or-v1-...), понимает, с какого баланса нужно списать деньги, перенаправляет запрос к нужной модели (например, к той же Claude 3.5 Sonnet) и возвращает ответ обратно в ваш терминал.
> 💰 **Итог:** Для Anthropic вы вообще не существуете как клиент. Все финансовые расчеты происходят только между вами и OpenRouter. С вашего баланса на OpenRouter списывается ровно столько центов, сколько токенов потратил Claude Code за конкретный текстовый запрос. Если на OpenRouter баланс $0 — Claude Code просто выдаст ошибку сети.
> 
## 🛠️ Как технически подменить «трубу» на Макбуке?
Чтобы Claude Code работал через OpenRouter и не требовал аккаунта Anthropic, вам нужно прописать три переменные окружения в конфигурационный файл вашего терминала на Макбуке (обычно это файл ~/.zshrc).
Делается это очень просто:
 1. Открываете терминал и вводите команду для редактирования файла конфигурации:
   ```bash
   nano ~/.zshrc
   
   ```
 2. Листаете в самый низ файла и вставляете туда следующие строки:
   ```bash
   # Перенаправляем Claude Code на OpenRouter
   export ANTHROPIC_BASE_URL="https://openrouter.ai/api"
   export ANTHROPIC_API_KEY="" # Оставляем пустым, чтобы не было конфликтов
   export ANTHROPIC_AUTH_TOKEN="ваш_ключ_от_openrouter_сюда"
   
   ```
   *(Обратите внимание: адрес должен быть именно https://openrouter.ai/api без суффикса /v1, так как Claude Code общается на родном языке Anthropic, который OpenRouter понимает напрямую).*
 3. Нажимаете Ctrl + O, затем Enter (для сохранения) и Ctrl + X (для выхода).
 4. Перезапускаете терминал командой:
   ```bash
   source ~/.zshrc
   
   ```
Теперь, когда вы перейдете в папку со своим проектом и запустите claude, утилита автоматически подключится к OpenRouter, используя ваш единый долларовый баланс. Вы сможете спокойно писать код, запускать агентов, а в личном кабинете OpenRouter наглядные графики покажут вам, сколько именно центов ушло на каждую сессию кодинга.

# 🎨 Цветовые палитры — исследование

> Цель: бизнес-сайт, ощущение **доверие + премиум**

---

## ✅ Понравились варианты 4 и 5

### Вариант 4 — Медь (тёплый, органичный)

| Элемент | HEX |
|---|---|
| Фон | `#0D1B2E` тёмно-синий |
| Текст | `#F5ECD7` тёплый кремовый |
| Акцент / кнопки | `#B87333` медь |
| Второстепенный | `#6B7280` серый |

Характер: теплее, органичнее, уютный статус

---

### Вариант 5 — Фиолетовый градиент (современный, необычный)

| Элемент | HEX |
|---|---|
| Фон | `#0D1130` → пурпур (градиент) |
| Текст | `#F5ECD7` кремовый |
| Акцент / детали | `#D4A017` золото |
| Кнопки | `#4F46E5` индиго |

Характер: таинственный, глубокий, необычный

## Базовая скомбинированная палитра

| Элемент | HEX |
|---|---|
| Фон | `#0D1B2E` |
| Акцент синий | `#2563EB` |
| Золото | `#D4A017` |
| Текст светлый | `#E8F0FF` |
| Текст кремовый | `#F0EBE0` |

---

## Правила применения

- Не больше 3 цветов — фон, основной, акцент
- Контраст текста минимум **4.5:1** (WCAG)
- Кнопки — всегда акцентный цвет
- Белое пространство = премиум-ощущение

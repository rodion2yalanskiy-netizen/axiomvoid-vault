## 🤖 Все AI-агенты системы

### 1. 🐍 Telegram Бот (bot.py + analyzer.py)

**Тип:** Python (python-telegram-bot) на Railway cloud
**Запуск:** Постоянный процесс. Railway project `62add92f` / service `56ff4171`.

**Режим заметок:** текст → classify_note() → Claude Sonnet via OpenRouter → предлагает vault/папку → GitHub API PUT → vault

**Режим Reels:** Instagram URL → yt-dlp → ffmpeg → Groq Whisper Large v3 → Claude анализ → Perplexity исследование → задача в Задачи/ status:delegated

**Режим чата:** многоходовой диалог через OpenRouter + анализ фото плитки

**Ограничения:**
- Все vault'ы → один репо qsnera-vault (Цифровой мозг и Личная жизнь тоже там)
- Сессии в /tmp/bot_sessions.json — TTL 1ч, переживают рестарт Railway
- Groq Whisper: max ~25MB аудио, язык жёстко русский

### 2. ⚙️ Local Agent (local-agent.sh)

**Тип:** Bash, launchd com.qsnera.local-agent
**Запуск:** Каждые 5 минут (StartInterval: 300), RunAtLoad

**Алгоритм:**
1. Ротация лога + heartbeat
2. sort_tasks_by_priority(): urgent → normal → low
3. До 5 задач за раз, до 2 параллельно (MAX_PARALLEL=2)
4. tool:code → claude --print (таймаут 20 мин, background+kill)
5. tool:antigravity → AppleScript → clipboard → Antigravity.app
6. Создаёт Отчёты/Отчёт - НАЗВАНИЕ.md
7. git add -A && commit && push

**⚠️ Обнаруженная проблема:** DELEGATE_DIR="$VAULT/Задачи" (строка 17), но в vault существуют ДВЕ папки: `Задачи/` и `📝 Задачи/` — задачи из второй папки не выполняются!

## 📈 Точки роста и проблемы

### 🔴 Критические (исправить срочно)

**1. Двойные папки задач**
- Существуют и `Задачи/` и `📝 Задачи/`
- local-agent читает ТОЛЬКО `Задачи/`
- Задачи в `📝 Задачи/` никогда не выполняются!
- Решение: проверить bot.py путь, мигрировать файлы, удалить `📝 Задачи/`

**2. TELEGRAM_TOKEN в plist**
- `com.qsnera.local-agent.plist` содержит TOGETHER_API_KEY открытым текстом
- Видим через `launchctl print`
- Решение: читать только из `~/.claude/.env`

### 🟡 Потенциальные проблемы

**3. Antigravity AppleScript хрупко**
- Клик по координатам `winHeight - 60` — сломается при смене UI
- Не работает при заблокированном экране

**4. Цифровой мозг пушится каждые 5 мин**
- Лог: Brain push каждые ~7 мин (2 файла)
- Триггерит GitHub Actions → API лимиты

**5. Рабочая память (working/) не заполняется**
- session-start.sh читает `working/ДАТА.md`
- Но ни один скрипт её не создаёт
- Решение: добавить запись в working/ в session-stop.sh

**6. 8 GitHub Actions workflows — но документированы только 2**
- Статус code-reviewer, qa-tester, router-agent, sentinel неизвестен

### 🟢 Идеи улучшений

**7. Веб-хук вместо polling**
- Сейчас задержка 5 мин (vault-sync polling)
- GitHub webhook → Railway → bot.py → немедленная обработка → <30 сек

**8. Прогресс задачи в реальном времени**
- TG: "⏳ Задача взята в работу..." при старте выполнения

**9. Retry при timeout**
- Добавить retry_count в frontmatter
- При timeout: retry_count-- → при 0 → fail с объяснением

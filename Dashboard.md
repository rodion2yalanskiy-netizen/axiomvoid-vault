## ⚡ Как создать задачу

Напиши в Telegram что нужно — бот сам создаст заметку в нужном месте.

Или напрямую в файл `Задачи/Название.md`:
```
---
status: delegated
tool: code
task_name: Название
---
Описание задачи
```

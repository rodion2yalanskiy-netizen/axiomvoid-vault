# 📝 Шаблоны задач QSNera

Скопируй нужный шаблон в папку `🤖 AI Задачи/` и заполни.
Пайплайн запустится автоматически через 5 минут.

---

## 🔧 Выбор инструмента (`tool:`)

| tool | Что делает | Когда использовать |
|------|-----------|-------------------|
| `openrouter` | Claude Sonnet 4.5 (текст) | Анализ, написание, планирование |
| `vision` | Claude с поддержкой изображений | Анализ скриншотов, дизайна |
| `images` | FLUX (Together AI) | Генерация изображений |
| `code` | Claude Code локально | Правки кода, сайта, git |
| `antigravity` | Antigravity/Gemini | Computer Use, браузер |

---

## Шаблоны ниже:

- `шаблон-текст.md` — текстовая задача (по умолчанию)
- `шаблон-vision.md` — анализ изображений
- `шаблон-images.md` — генерация изображений
- `шаблон-code.md` — правки кода и сайта
- `шаблон-antigravity.md` — компьютер use, браузер

# ⚠️ АРХИТЕКТУРНЫЕ ПРАВИЛА Axiom:Void Vault

> **ОБЯЗАТЕЛЬНО для всех агентов, ботов и скриптов.**
> Нарушение этих правил ломает синхронизацию iOS Obsidian Sync.

---

## 🚫 КАТЕГОРИЧЕСКИ ЗАПРЕЩЕНО

### 1. Папки с эмоджи в именах

```
❌ НЕЛЬЗЯ создавать:
  📝 Задачи/
  ✅ Выполнено Claude/
  🤖 AI Задачи/
  👥 Клиенты/
  💡 Маркетинг/
  🏗 Сайт/
  📊 Отчёты/
  [любая папка с эмоджи]
```

**Почему:** iOS Obsidian Sync падает на папках с Unicode emoji (U+1F000+).
Эмоджи в **именах файлов** — допустимо. Эмоджи в **именах папок** — нет.

### 2. Создание новых папок без согласования

Нельзя создавать папки не из списка разрешённых. Если нужна новая папка — напиши задачу для Родиона.

### 3. Изменение структуры vault'а без `[skip ci]`

Все автоматические коммиты должны содержать `[skip ci]` чтобы не запускать GitHub Actions рекурсивно.

## 📋 ПРАВИЛА ИМЕНОВАНИЯ ФАЙЛОВ

```
Отчёты:    "Отчёт - Название задачи.md"   ← дефис " - ", не двоеточие!
Задачи:    "Название задачи.md"
Клиенты:   "Имя Фамилия.md"
```

**Разделитель в отчётах:** ` - ` (пробел-дефис-пробел), **не** `:` и **не** эмоджи.

## 🤖 ПРАВИЛА ДЛЯ КАЖДОГО АГЕНТА

### Telegram Bot (Railway)
- ✅ Сохранять заметки ТОЛЬКО через ALLOWED_FOLDERS
- ✅ Очищать эмоджи из имён папок перед сохранением
- ✅ Задачи — только в `Задачи/`
- ❌ НЕЛЬЗЯ создавать новые папки через GitHub API

### GitHub Actions (`ai-agent.yml`)
- ✅ Читать задачи из `Задачи/*.md`
- ✅ Сохранять отчёты в `Отчёты/`
- ❌ НЕЛЬЗЯ создавать `✅ Выполнено Claude/`
- ❌ НЕЛЬЗЯ создавать `📝 Задачи/`
- ✅ Все коммиты с `[skip ci]`

### Local Agent (`local-agent.sh`)
- ✅ Читать из `$VAULT/Задачи/`
- ✅ Писать отчёты в `$VAULT/Отчёты/`
- ✅ `git add -A` всегда (не `git add <path>`)
- ✅ `git pull --rebase` перед push

### Claude Code (сессии)
- ✅ При создании файлов — проверять путь на эмоджи в папках
- ✅ Работать через `~/vaults/Бизнес Axiom:Void/` (с пробелом!)
- ❌ НЕЛЬЗЯ создавать папки вне списка разрешённых
- ✅ Коммиты с `[skip ci]` если не нужно запускать Actions

### Vault Sync (`vault-sync.sh`)
- Только синхронизация, не создаёт папки

## 🛡 Guardian (автоматическая защита)

Скрипт `~/.claude/agents/vault-guardian.sh` запускается каждые 10 минут и:
1. Ищет папки с эмоджи в vault'е
2. Перемещает файлы в правильные места
3. Удаляет пустые эмоджи-папки
4. Отправляет Telegram уведомление если нашёл нарушение

## 🔧 ЕСЛИ НАРУШИЛ — КАК ИСПРАВИТЬ

```bash
# 1. Перенести файлы из emoji-папки в правильную
mv ~/vaults/Бизнес\ Axiom:Void/✅\ Выполнено\ Claude/*.md \
   ~/vaults/Бизнес\ Axiom:Void/Отчёты/

# 2. Удалить пустую emoji-папку
rmdir ~/vaults/Бизнес\ Axiom:Void/✅\ Выполнено\ Claude/

# 3. Закоммитить
cd ~/vaults/Бизнес\ Axiom:Void
git add -A
git commit -m "fix: удалить emoji-папки [skip ci]"
git pull --rebase && git push
```

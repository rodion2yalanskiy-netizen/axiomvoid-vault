# ✅ Задачи (Tasks)

Папка для конкретных задач с быстрым выполнением через OpenRouter.

## Как использовать

1. Создай файл `МояЗадача.md`
2. Напиши задачу (без frontmatter — тоже работает)
3. Через 5-10 мин появится отчёт в `✅ Выполнено Claude/`

## Отличие от AI Задачи

- `tasks/` → **Task Executor** (быстро, любые задачи)
- `🤖 AI Задачи/` → **AI Agent** (поддерживает tool:code, vision, images)

## Пример

```markdown
Напиши 5 идей для постов в Instagram для студии укладки премиум-плитки QSNera.
Тема: осенние интерьерные тренды 2026.
```

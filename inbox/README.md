# 📥 Входящие (Inbox)

Кидай сюда любые заметки — роутер сам определит тему и переместит.

## Как использовать

1. Создай `.md` файл в этой папке
2. Напиши что угодно (заметку, идею, задачу)
3. Через 5-10 мин роутер классифицирует и переместит:
   - **Бизнес** → `🤖 AI Задачи/` (выполнится AI)
   - **Задача** → `tasks/` (выполнится через Task Executor)
   - **Техника** → `📋 AI Делегат/` (выполнит Claude Code)
   - **Личное** → `📋 AI Делегат/` (переместится в Личная жизнь)

## Пример

```markdown
---
---
Нужно написать пост для Instagram про новый проект укладки мрамора в ЖК Triumph.
```

Роутер определит `business` и создаст задачу в AI Задачи.

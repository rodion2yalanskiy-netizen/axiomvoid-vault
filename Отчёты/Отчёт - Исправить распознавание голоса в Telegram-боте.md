## 🎯 Что было исправлено

### 1. Транскрипция OGG (главная проблема)
- **Причина:** Groq Whisper API требует явный mime-type для OGG/Opus файлов. Без него некоторые файлы отвергались
- **Фикс:** Добавлен словарь `MIME_TYPES` с правильными типами для всех форматов
- **Бонус:** Добавлен `language='ru'` — подсказка Whisper что основной язык русский, повышает точность на 20–30%

```python
MIME_TYPES = {
    ".ogg":  "audio/ogg",   # Telegram голосовые
    ".opus": "audio/ogg",
    ".mp3":  "audio/mpeg",
    ".m4a":  "audio/mp4",
    ...
}
```

### 2. Структура хранилищ (параллельно исправлено)
- **Причина:** `classify_note()` возвращал несуществующие папки ("Знания", "Автономный доход") → бот создавал новые папки с emoji в vault'е → ломал iOS Obsidian Sync
- **Фикс:** Добавлен `ALLOWED_FOLDERS` с жёсткой валидацией + автоочистка emoji из всех полей
- **Защита:** Двойная валидация в `save_note_to_obsidian()` 

### 3. Очистка vault'а от emoji-папок
Удалены все папки с emoji из всех vault'ов (ломали Obsidian Sync на iPhone):
- `✅ Выполнено Claude/` → содержимое перенесено в `Отчёты/`

### 4. Технические файлы убраны из vault'а
